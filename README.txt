CJ스튜디오 관리자 기능 업데이트

추가 기능
1. 신청곡 승인
2. 신청곡 거절
3. 신청곡 완전 삭제
4. 승인된 일반 음원의 '유튜브 사운드 확인' 버튼
   - youtubeUrl / youtubeLink / videoUrl 필드가 있으면 해당 영상 열기
   - 링크가 없으면 곡명 + 가수명으로 유튜브 검색 열기

적용 방법
1. 압축을 풉니다.
2. admin-app.js와 admin-style.css를 GitHub 저장소 루트에 덮어씁니다.
3. Commit changes를 누릅니다.
4. GitHub Pages가 갱신된 뒤 관리자 페이지를 새로고침합니다.
