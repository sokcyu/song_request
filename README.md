# CJ스튜디오 가사 키워드 전용 신청 기능 추가본

## 신청 방식
1. 노래 제목·가수로 신청
2. 가사 키워드로 신청

## 가사 키워드 신청
- 노래 제목과 가수 입력 불필요
- 가사 키워드 1개 이상 필수
- 최대 5개까지 입력
- 쉼표로 구분
- 음악 키워드 5가지 중 하나 선택
- 관리자 화면에는 `가사 키워드 신청`으로 표시

## 기존 기능
- 승인 시 `음원 승인`
- 거절 시 `음원 거절`
- 관리자 초기 비밀번호 `1234`
