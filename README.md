## 변경 사항
- 이용자와 게스트 신청 화면에 스크롤 가능한 장르 선택 목록 추가
- 장르 선택 필수
- 관리자 화면에 선택 장르 표시
- 가사 키워드 신청과 제목·가수 신청 모두 지원
- 승인 시 `음원 승인`
- 거절 시 `음원 거절`
- 관리자 초기 비밀번호 `1234`
