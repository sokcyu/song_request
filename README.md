# CJ스튜디오 가사 키워드 신청 기능 추가본

## 음악 장르 키워드 5가지
- K-POP
- POP
- 발라드
- 댄스
- 트로트

## 가사 키워드 신청
- 이용자와 게스트가 가사에 넣을 단어 또는 주제를 직접 입력
- 쉼표로 구분하여 최대 5개
- 키워드 하나당 최대 20자
- 예: 사랑, 바다, 추억, 여름, 여행
- 관리자 신청곡 목록에 가사 키워드 표시

## 기존 기능
- 승인 시 `음원 승인`
- 거절 시 `음원 거절`
- 관리자 초기 비밀번호 `1234`
