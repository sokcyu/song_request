# CJ스튜디오 승인 표시 변경 통합본

## 변경 사항
- 관리자가 승인하면 이용자와 게스트 화면에 `음원 승인` 표시
- 관리자가 거절하면 이용자와 게스트 화면에 `음원 거절` 표시
- 승인/거절 결과를 Firestore에서 실시간 반영
- 이용자와 게스트가 이 기기에서 신청한 곡의 처리 상태 확인
- 관리자 초기 비밀번호: 1234

## 업로드할 파일
저장소 루트에 이 ZIP의 모든 파일을 업로드하고 Commit changes를 누르세요.

## 중요
관리자 페이지가 Firebase Authentication 없이 공용 비밀번호 방식이므로,
이 프로젝트의 Firestore 규칙은 공개 읽기/쓰기를 사용합니다.
관리자 링크와 비밀번호를 외부에 공개하지 마세요.
